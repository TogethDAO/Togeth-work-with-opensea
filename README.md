# **opensea-sdk**
TogethDAO works with opensea sdk to generate calldata for contract interaction. 

# Installation

```
npm install
```

# Configurate Environment variables

You need to create an '.env' file to configurate 
```
privateKey
rinkeby provider url
mainner provider url
opensea api key
```
